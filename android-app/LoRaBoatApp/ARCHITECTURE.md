## Device Connection Flow

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ Connection Flow                                  в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

1. User clicks "Connect Device"
   в””в”Җ Start DeviceListActivity

2. Scan for paired Bluetooth devices
   в””в”Җ Call BluetoothAdapter.getBondedDevices()

3. Display device list
   в”ңв”Җ HC-05 (00:19:5D:AB:C1)
   в”ңв”Җ LoRa-BT (20:16:D8:50:F0)
   в””в”Җ BlueMod (98:D3:31:20:50)

4. User selects device
   в””в”Җ Pass device address to MainActivity

5. Create Bluetooth socket
   в””в”Җ device.createRfcommSocketToServiceRecord(UUID)

6. Connect (in background thread)
   в””в”Җ socket.connect()

7. Get input/output streams
   в”ңв”Җ inputStream = socket.getInputStream()
   в””в”Җ outputStream = socket.getOutputStream()

8. Start read thread
   в””в”Җ ReadThread.start()

9. Connection callback
   в””в”Җ onConnected() triggered

10. Display "Connected" status
    в””в”Җ Button changes to "Disconnect"

11. Ready to receive data
    в””в”Җ App waits for Bluetooth messages
```
